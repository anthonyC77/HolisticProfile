# Maison 14 — Ressource Universelle

**Formule :** M14 = nombre AVANT réduction lors du calcul de M5, converti en Arcane Mineur
**Autre nom :** Roue de secours, Arcane Mineur de synthèse

## Ce que cette maison révèle
La Maison 14 est unique dans le Référentiel : elle porte un Arcane Mineur, pas un Arcane Majeur. Là où les maisons 1–13 décrivent des énergies profondes et structurelles, M14 indique une ressource concrète, pratique, accessible au quotidien. C'est la "roue de secours" — ce qui est toujours disponible, même quand tout le reste semble absent.

## L'arcane mineur dans cette maison indique
Une ressource concrète à mobiliser. La famille (Bâtons, Coupes, Épées, Deniers) dit le domaine ; la carte (Roi, Reine, Cavalier, Valet, As, 2-10) dit la façon de l'utiliser.

## Les quatre familles et leur énergie

**Bâtons (1–14) — Feu, Action, Création**
L'énergie du faire, de l'initiative, de la créativité active. La ressource est dans l'action, la mise en mouvement, la création concrète.

**Coupes (15–28) — Eau, Émotions, Relations**
L'énergie du ressentir, de la relation, de l'intuition émotionnelle. La ressource est dans les liens, l'écoute, la profondeur émotionnelle.

**Épées (29–42) — Air, Mental, Clarté**
L'énergie du penser, du trancher, de la clarté intellectuelle. La ressource est dans la pensée, le discernement, la parole juste.

**Deniers (43–56) — Terre, Corps, Matière**
L'énergie du concret, du corps, de l'ancrage. La ressource est dans le tangible, le pratique, le corps, l'argent, la terre.

## Les figures de cour
- **Roi** : maîtrise, autorité dans ce domaine — incarner pleinement cette énergie
- **Reine** : sagesse intérieure, réceptivité active dans ce domaine
- **Cavalier** : mouvement, quête, élan dans ce domaine — partir vers
- **Valet** : apprentissage, curiosité, disponibilité dans ce domaine
- **As** : graine, potentiel pur — l'essence même de cette famille à l'état le plus pur

## Questions à poser en séance
- Quelle ressource oubliez-vous de mobiliser quand vous en avez besoin ?
- Comment utilisez-vous naturellement l'énergie de cette famille (feu/eau/air/terre) ?
- En quoi cette carte décrit-elle une ressource concrète disponible pour vous ?
- Y a-t-il une façon d'agir dans ce domaine que vous sous-utilisez ?

## Note clinique
M14 est souvent reçue avec légèreté — "juste un arcane mineur" — mais elle est précieuse cliniquement : elle donne une indication concrète, pratique, sur ce que la personne peut mobiliser maintenant. À utiliser pour terminer une séance sur une note d'action possible.
