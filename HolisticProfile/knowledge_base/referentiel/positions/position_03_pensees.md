# Maison 3 — Pensées / Désirs / Peurs

**Formule :** M3 = somme des chiffres de l'ANNÉE de naissance (réduit base 22)

## Ce que cette maison révèle
Le monde intérieur : les pensées profondes, les désirs non exprimés, les peurs fondamentales. Ce qui se passe dans la tête et le cœur de la personne, souvent à son insu.
