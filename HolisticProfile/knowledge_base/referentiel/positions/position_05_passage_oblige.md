# Maison 5 — Passage Obligé

**Formule :** M5 = M1 + M2 + M3 + M4 (réduit base 22)
**Autre nom :** Épreuve incontournable, Transformation nécessaire

## Ce que cette maison révèle
L'épreuve que la personne ne peut pas éviter, le passage de transformation qui reviendra sous différentes formes jusqu'à ce qu'il soit traversé consciemment. Ce n'est pas un obstacle arbitraire — c'est le cœur du chemin.
